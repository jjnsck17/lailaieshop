## 独享和共享到底有什么区别

这是很多人关心的问题。我刚开始也搞不清楚。

**共享账号**，就是多个人（一般是4-5个人）共用一个Netflix主账号，每个人有自己的Profile（子账号），可以添加自己的待看列表和观影记录。这种方式最大的优点是便宜，但缺点也很明显：

1. 可能被其他用户"挤掉"
2. 观看高峰期会受影响
3. Netflix严打共享后，容易出问题

**独享账号**，就是你一个人独占一个完整账户。可以设置自己的密码、邮箱，有完整访问权限，想看什么看什么，不受他人影响。价格会贵一些，但稳定性和使用体验都好很多。
